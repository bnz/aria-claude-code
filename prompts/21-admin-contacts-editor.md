# Промпт 21: CMS — Редактор Contacts

## Контекст
Translations editor готов (промпт 20). Создаём редактор контактов.

## Задача
Создай редактор для раздела Contacts:

1. **Загрузка**: `contacts.{en,lv,ru}.json`
2. **Поля формы**:
   - `phone` — общий для всех языков (или раздельный — реши, зафиксируй в DECISIONS.md)
   - `address` — по языкам (табы en/lv/ru)
   - `mapEmbedUrl` — общий (одна карта для всех языков)
   - `introText` — по языкам
   - `workHours` — по языкам
3. **Превью карты**: показать iframe предпросмотр рядом с полем mapEmbedUrl
4. **Валидация**: Zod (ContactsSchema)
5. **Автосохранение в draft**

## Требования к тестам
- Форма загружает и отображает контактные данные
- Валидация: телефон не может быть пустым, адрес минимум 3 символа
- Изменения сохраняются в draft
- Запусти тесты

## Ожидаемый результат
Удобный редактор контактов с превью карты и валидацией.
