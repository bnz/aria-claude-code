# Промпт 10: Страница «Контакты» (/contacts)

## Контекст
About-страница готова (промпт 09). Создаём страницу контактов.

## Задача
Создай страницу `app/[lang]/contacts/page.tsx`:

1. Загрузка данных из `contacts.{lang}.json`
2. Кликабельный телефон (`<a href="tel:...">`)
3. Адрес (текстовый, не картинка — для SEO)
4. Часы работы
5. Вводный текст (introText)
6. Встроенная карта Google Maps (iframe embed из `mapEmbedUrl`)
7. CTA-кнопка «Позвонить» — крупная, заметная
8. Mobile-first: на мобильном CTA звонка должен быть максимально доступен

## Ссылки на спек
- Секция 3.1 (`/contacts`)
- Секция 14.4 (Local SEO — NAP consistency)
- Секция 15.4 (Contacts schema)

## Требования к тестам
- Страница рендерится для каждого языка
- Телефон — кликабельная ссылка `tel:`
- Адрес отображается текстом
- MapEmbedUrl рендерит iframe (если указан)
- Запусти тесты

## Ожидаемый результат
Страница контактов, оптимизированная для конверсии (звонок) и Local SEO.
