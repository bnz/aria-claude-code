# Промпт 06: Layout, Header, Footer

## Контекст
Маршрутизация настроена (промпт 05). Создаём общий каркас публичного сайта.

## Задача
Создай общий layout с header и footer для публичного сайта:

1. **Header** (`src/components/Header.tsx`):
   - Логотип/название (текстовый)
   - Навигация: Главная, Информация, Статьи, Состояния, О специалисте, Контакты
   - Переключатель языков (из промпта 05)
   - Мобильное меню (бургер) — mobile-first!
   - Все тексты из `translations`

2. **Footer** (`src/components/Footer.tsx`):
   - Копирайт с годом
   - Контактная информация (телефон — кликабельный)
   - Навигационные ссылки
   - Все тексты из `translations`

3. **Layout** (`app/[lang]/layout.tsx`):
   - Header + main + Footer
   - Metadata (базовые мета-теги)

4. **Тема**: поддержка light/dark через `prefers-color-scheme` (CSS/Tailwind `dark:` классы), БЕЗ переключателя

## Ссылки на спек
- Секция 3.1 (Разделы/страницы)
- Секция 4 (Mobile-First)
- Правила работы: темы — авто light/dark, без переключателя

## Требования к тестам
- Рендеринг Header с навигацией на каждом языке
- Мобильное меню открывается/закрывается
- Footer показывает кликабельный телефон
- Dark mode применяет правильные CSS классы
- Запусти тесты

## Ожидаемый результат
Полноценный responsive layout с header, footer, мобильным меню и поддержкой тёмной темы.
