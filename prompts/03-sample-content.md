# Промпт 03: Контент-заглушки (sample data)

## Контекст
Zod-схемы созданы (промпт 02). Нужны тестовые данные для разработки публичного сайта.

## Задача
Создай JSON-файлы с реалистичными заглушками для всех типов контента:

1. `translations.{en,lv,ru}.json` — UI строки (навигация, кнопки, footer)
2. `contacts.{en,lv,ru}.json` — тестовые контакты (Рига, телефон, часы работы)
3. `info.{en,lv,ru}.json` — 3-4 секции про акупунктуру (text, bullets, image)
4. `about.{en,lv,ru}.json` — профиль специалиста с credentials и сертификатами
5. `articles/index.json` + 2 тестовые статьи с slug-папками и `article.{en,lv,ru}.json`
6. `conditions/index.json` + 3 тестовых состояния с slug-папками и `condition.{en,lv,ru}.json`

Контент должен быть на соответствующих языках (не lorem ipsum, а осмысленный текст про акупунктуру). imagePath-поля указывают на `/media/placeholder.jpg`.

## Требования к тестам
- Напиши тест, который загружает каждый JSON и валидирует его через соответствующую Zod-схему
- Проверь, что все `id` и `slug` совпадают между языками
- Запусти тесты и убедись, что они проходят

## Ожидаемый результат
Полный набор JSON-файлов с контентом на 3 языках, прошедших Zod-валидацию.
