# Промпт 04: Content Loader (утилиты загрузки данных для SSG)

## Контекст
Есть Zod-схемы (промпт 02) и JSON-контент (промпт 03). Нужен слой загрузки данных для Next.js SSG.

## Задача
Создай утилиты для загрузки и валидации контента на этапе build:

1. `src/lib/content.ts` — функции для чтения JSON из `/content`:
   - `getTranslations(lang)` → загрузка + Zod-валидация
   - `getContacts(lang)` → загрузка + Zod-валидация
   - `getInfo(lang)` → загрузка + Zod-валидация
   - `getAbout(lang)` → загрузка + Zod-валидация
   - `getArticlesList()` → загрузка index.json + валидация
   - `getArticle(slug, lang)` → загрузка + Zod-валидация
   - `getConditionsList()` → загрузка index.json + валидация
   - `getCondition(slug, lang)` → загрузка + Zod-валидация
2. Все функции должны бросать понятные ошибки при невалидных данных
3. Создай хелпер `getAllLanguageVersions<T>(loader)` для загрузки всех 3 языков сразу
4. Создай `src/lib/languages.ts` — константы языков, типы, хелпер для `generateStaticParams`
