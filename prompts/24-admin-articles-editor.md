# Промпт 24: CMS — Редактор Articles (CRUD)

## Контекст
About editor готов (промпт 23). Создаём редактор статей — один из ключевых разделов CMS.

## Задача
Создай полный CRUD для статей:

1. **Список статей**:
   - Загрузка из `articles/index.json` + заголовки статей
   - Таблица/список: slug, title (первого языка), published, order
   - Кнопки: Создать, Редактировать, Удалить
   - Переупорядочивание (drag & drop или ↑↓)

2. **Создание статьи**:
   - Ввод slug (валидация: lowercase, no spaces, unique)
   - Автогенерация id (uuid или slug-based)
   - Создание пустых `article.{en,lv,ru}.json`
   - Добавление записи в `index.json`

3. **Редактирование статьи**:
   - Табы по языкам (en/lv/ru)
   - Поля: title, excerpt, heroImagePath (Media Library), seo, sections
   - Секции: как в Info editor (text, image) — переиспользуй компонент
   - Валидация Zod (ArticleSchema)

4. **Удаление статьи**:
   - Подтверждение: «Удалить статью XYZ на всех языках?»
   - Удаление из index + всех language файлов (синхронно!)

5. **Toggle published**: быстрое вкл/выкл прямо из списка
6. **Автосохранение в draft**

## Требования к тестам
- CRUD: создание, чтение, обновление, удаление
- Slug уникальность
- Удаление синхронно для всех языков
- Published toggle работает
- Валидация excerpt (max 280 символов)
- Запусти тесты

## Ожидаемый результат
Полный CRUD редактор статей с мультиязычностью и валидацией.
