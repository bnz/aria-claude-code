# Промпт 15: Оптимизация производительности и изображений

## Контекст
Публичный сайт полностью создан (промпты 07-14). Оптимизируем для Core Web Vitals.

## Задача
Оптимизируй производительность публичного сайта:

1. **Изображения**:
   - Используй `next/image` для всех изображений
   - Настрой sizes и priority для hero-изображений (LCP)
   - Добавь placeholder blur (если возможно при SSG)
   - Создай компонент-обёртку `OptimizedImage` для единообразной работы с `/media/` путями

2. **Шрифты**:
   - Используй `next/font` для загрузки шрифтов
   - `font-display: swap` для предотвращения CLS

3. **CSS/JS**:
   - Убедись, что код админки НЕ попадает в бандл публичных страниц
   - Минимизируй блокирующие ресурсы

4. **Core Web Vitals**:
   - LCP < 2.5s
   - CLS < 0.1
