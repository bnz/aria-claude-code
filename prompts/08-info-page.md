# Промпт 08: Страница «Информация» (/info)

## Контекст
Главная страница готова (промпт 07). Создаём страницу Info.

## Задача
Создай страницу `app/[lang]/info/page.tsx`:

1. Загрузка данных из `info.{lang}.json`
2. Рендеринг секций по типам (discriminated union из InfoSectionSchema):
   - `text` — заголовок + текстовый блок
   - `bullets` — заголовок + список пунктов
   - `image` — изображение с подписью
3. Создай переиспользуемый компонент `ContentSections` для отображения секций (он пригодится и для conditions)
4. Mobile-first вёрстка, адаптивные изображения

## Ссылки на спек
- Секция 3.1 (`/info`)
- Секция 15.5 (Info schema)

## Требования к тестам
- Страница рендерится для каждого языка
- Каждый тип секции корректно отображается
- Компонент `ContentSections` работает с разными наборами секций
- Запусти тесты

## Ожидаемый результат
Страница Info с динамическим рендерингом секций разных типов.
