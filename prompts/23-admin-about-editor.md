# Промпт 23: CMS — Редактор About

## Контекст
Info editor готов (промпт 22). Создаём редактор для страницы «О специалисте».

## Задача
Создай редактор для раздела About:

1. **Загрузка**: `about.{en,lv,ru}.json`
2. **Поля**:
   - `title` — по языкам
   - `summary` — textarea по языкам
   - `seo` — SEO блок по языкам
   - `experienceYears` — число (общее для всех языков)
   - `credentials` — динамический список строк по языкам (добавить/удалить пункт)
   - `certificates` — динамический список объектов:
     - `title` — по языкам
     - `imagePath` — выбор из Media Library (кнопка → модальное окно)

3. **Валидация**: Zod (AboutSchema)
4. **Автосохранение в draft**

## Ссылки на спек
- Секция 8 (пункт 6 — About)
- Секция 15.6 (AboutSchema)

## Требования к тестам
- Добавление/удаление credentials
- Добавление/удаление certificates
- Валидация числового поля experienceYears
- Изменения сохраняются в draft
- Запусти тесты

## Ожидаемый результат
Редактор About с E-E-A-T фокусом (квалификации, сертификаты, опыт).
