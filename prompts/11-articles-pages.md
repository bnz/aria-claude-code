# Промпт 11: Страницы статей (/articles)

## Контекст
Базовые страницы готовы (промпты 07-10). Создаём раздел статей.

## Задача
Создай страницы для раздела статей:

1. **Список статей** `app/[lang]/articles/page.tsx`:
   - Загрузка `articles/index.json` + данные каждой статьи
   - Карточки: заголовок, excerpt, hero-изображение (если есть)
   - Только опубликованные (`published: true`)
   - Сортировка по `order`
   - Ссылки на `/[lang]/articles/[slug]`

2. **Страница статьи** `app/[lang]/articles/[slug]/page.tsx`:
   - `generateStaticParams` — все slug × все языки
   - Загрузка `article.{lang}.json`
   - Hero-изображение
   - Заголовок + секции (text, image) через компонент `ContentSections` или аналогичный
   - Навигация: «Назад к списку статей»

3. Mobile-first вёрстка для обоих страниц

## Ссылки на спек
- Секция 3.1 (`/articles`, `/articles/[slug]`)
- Секция 15.7 (Articles schema)

## Требования к тестам
- Список показывает только published статьи
- Страница статьи рендерит все секции
- generateStaticParams возвращает правильные комбинации lang × slug
- Запусти тесты и `next build`

## Ожидаемый результат
Полноценный раздел статей со списком и детальными страницами.
