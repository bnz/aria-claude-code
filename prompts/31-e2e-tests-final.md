# Промпт 31: Финальные E2E тесты и интеграционная проверка

## Контекст
Весь проект реализован (промпты 01-30). Финальная проверка.

## Задача
E2E фреймворк уже настроен с промпта 01, и E2E тесты писались на каждом шаге. Теперь — финальная ревизия покрытия и интеграционная проверка:

1. **Public site E2E** — убедись, что покрыты все сценарии:
   - Главная загружается на каждом языке
   - Навигация между всеми страницами
   - Переключение языков сохраняет текущую страницу
   - CTA кнопки содержат `tel:` ссылки
   - Статьи и conditions: переход к деталям и обратно
   - Dark mode (prefers-color-scheme) применяет стили
   - Мобильный viewport: бургер-меню работает

2. **SEO проверка**:
   - Каждая страница имеет уникальный title
   - hreflang теги присутствуют
   - JSON-LD валиден
   - sitemap.xml содержит все URL

3. **CMS E2E** (если возможно с mock GitHub):
   - Login → Dashboard
   - Редактирование контента → сохранение в draft
   - Publish flow (mock)

4. **Lighthouse проверка** (можно CI или ручная):
   - SEO score ≈ 100
   - Performance: LCP < 2.5s, CLS < 0.1

5. **Build проверка**:
   - `next build` проходит без ошибок и warnings
   - Static export генерирует корректное количество HTML файлов
