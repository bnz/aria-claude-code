# Промпт 05: Мультиязычная маршрутизация (i18n routing)

## Контекст
Content loader готов (промпт 04). Теперь настроим маршрутизацию с поддержкой 3 языков.

## Задача
Настрой мультиязычную маршрутизацию в Next.js App Router:

1. Создай структуру `app/[lang]/` для всех публичных страниц
2. Настрой `generateStaticParams` для генерации страниц на 3 языках (en, lv, ru)
3. Создай middleware или redirect с `/` на язык по умолчанию (en или определение из Accept-Language — реши и зафиксируй в DECISIONS.md)
4. Создай layout `app/[lang]/layout.tsx`:
   - `<html lang={lang}>`
   - загрузка translations для текущего языка
   - передача translations через context или props
5. Создай компонент `LanguageSwitcher` — переключение между en/lv/ru (сохраняет текущую страницу)

- Проверь, что `generateStaticParams` возвращает все 3 языка
- Проверь, что LanguageSwitcher генерирует правильные ссылки
- Запусти `next build` и убедись, что страницы генерируются для каждого языка
- Запусти тесты
