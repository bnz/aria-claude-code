# Промпт 01: Инициализация проекта

## Контекст
Начинаем реализацию проекта по спецификации `acupuncture-spec.md`. Этот промпт — первый в серии.

## Задача
Инициализируй проект Next.js с App Router и настрой базовую конфигурацию:

1. Создай проект Next.js (App Router, TypeScript, TailwindCSS)
2. Настрой `next.config` для SSG (static export) и GitHub Pages
3. Создай структуру директорий `/content` согласно секции 6.1 спека
4. Создай пустую структуру `/public/media/`
5. Настрой базовый `tsconfig.json` с path aliases (`@/`)
6. Добавь зависимости: `zod` для валидации
7. Настрой ESLint и Prettier
8. Настрой E2E тестирование (Playwright): установи зависимости, создай `playwright.config.ts`, добавь `npm run test:e2e` в `package.json`
9. Настрой unit-тестирование (Jest или Vitest — реши и зафиксируй в DECISIONS.md): установи зависимости, создай конфиг, добавь `npm run test` в `package.json`
10. Создай файл `DECISIONS.md` для фиксации решений (согласно правилам работы в спеке)

## Требования к тестам
- Напиши unit-тест, что все обязательные директории `/content` существуют
- Напиши простой E2E тест, что dev-сервер запускается и отдаёт страницу
- Запусти `npm run test` и `npm run test:e2e` — оба должны пройти
- Запусти `npm run build` — билд без ошибок

## Ожидаемый результат
Чистый проект Next.js, готовый к разработке, со структурой папок по спеку, с настроенными unit- и E2E-тестами.
