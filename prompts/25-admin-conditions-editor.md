# Промпт 25: CMS — Редактор Conditions (CRUD)

## Контекст
Articles editor готов (промпт 24). Создаём редактор условий/симптомов — аналогичный, но с доп. полями.

## Задача
Создай полный CRUD для conditions (по аналогии с articles):

1. **Список conditions**:
   - Таблица: slug, title, published, order
   - Кнопки: Создать, Редактировать, Удалить
   - Переупорядочивание

2. **Создание condition**: slug + пустые JSON для 3 языков + запись в index

3. **Редактирование condition**:
   - Табы по языкам
   - Поля: title, intro, seo, sections (как Info), ПЛЮС:
   - `contraindications` — динамический список строк по языкам
   - `faq` — динамический список пар {q, a} по языкам (добавить/удалить вопрос-ответ)

4. **Удаление**: синхронно для всех языков + подтверждение

5. **Toggle published**
6. **Автосохранение в draft**

- CRUD операции
- FAQ редактор: добавление/удаление пар q/a
- Contraindications: добавление/удаление
- Удаление синхронно
- Валидация: min 1 секция, min 1 FAQ (если есть)
