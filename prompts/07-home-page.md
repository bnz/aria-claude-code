# Промпт 07: Главная страница

## Контекст
Layout готов (промпт 06). Создаём главную страницу.

## Задача
Создай главную страницу `app/[lang]/page.tsx`:

1. **Hero секция**: приветственный текст + CTA кнопка (звонок)
2. **Краткое описание**: что такое акупунктура (из info или отдельный блок)
3. **Превью контента**:
   - Последние статьи (2-3 карточки из articles)
   - Популярные состояния (3-4 карточки из conditions)
4. **CTA блок**: «Запишитесь на консультацию» + телефон
5. Все тексты — мультиязычные (из translations + контентных JSON)
6. Mobile-first вёрстка

## Ссылки на спек
- Секция 3.1 (`/` — главная)
- Секция 4 (Mobile-First)

## Требования к тестам
- Страница рендерится для каждого языка
- CTA кнопка содержит ссылку `tel:`
- Превью статей и условий отображаются
- Запусти тесты и `next build`

## Ожидаемый результат
Рабочая главная страница с hero, превью контента и CTA.
