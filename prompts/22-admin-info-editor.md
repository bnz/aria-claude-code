# Промпт 22: CMS — Редактор Info

## Контекст
Contacts editor готов (промпт 21). Создаём редактор для страницы «Информация».

## Задача
Создай редактор для раздела Info:

1. **Загрузка**: `info.{en,lv,ru}.json`
2. **Поля**:
   - `title` — по языкам (табы en/lv/ru)
   - `seo` — SEO поля (title, description, ogImagePath) по языкам
   - `sections` — динамический список секций

3. **Редактор секций** (самый сложный элемент):
   - Добавление секции: выбор типа (text / bullets / image)
   - Перетаскивание/изменение порядка секций (drag & drop или кнопки ↑↓)
   - Удаление секции
   - Для `text`: заголовок + текстовое поле (textarea/markdown)
   - Для `bullets`: заголовок + динамический список строк (добавить/удалить пункт)
   - Для `image`: выбор изображения (кнопка → открывает Media Library, промпт 25) + caption

4. **Синхронизация**: секции синхронизированы по индексу между языками (добавил секцию — она появляется во всех языках)
5. **Валидация**: Zod (InfoSchema)
6. **Автосохранение в draft**

- Добавление/удаление/переупорядочивание секций

- Синхронизация секций между языками
- Валидация: min 1 секция
- Запусти тесты
