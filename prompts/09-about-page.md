# Промпт 09: Страница «О специалисте» (/about)

## Контекст
Info-страница готова (промпт 08). Создаём страницу About (E-E-A-T).

## Задача
Создай страницу `app/[lang]/about/page.tsx`:

1. Загрузка данных из `about.{lang}.json`
2. Блок с фото специалиста (если есть)
3. Краткое описание (summary)
4. Опыт работы (experienceYears)
5. Список квалификаций (credentials) — список с иконками/маркерами
6. Сертификаты (certificates) — карточки с изображениями (если указан imagePath)
7. Mobile-first вёрстка
8. Это ключевая страница для E-E-A-T (доверие поисковиков), поэтому дизайн должен выглядеть профессионально

## Ссылки на спек
- Секция 3.1 (`/about`)
- Секция 15.6 (About schema)
- Секция 14.2 (E-E-A-T)

## Требования к тестам
- Страница рендерится для каждого языка
- Credentials и certificates отображаются
- Пустые опциональные поля не ломают рендеринг
- Запусти тесты

## Ожидаемый результат
Профессиональная страница «О специалисте» с акцентом на доверие.
